Android SDK Installation Setup:
---------------------------------

Step 1:
-------

Download the Android SDK application file from
https://greentechnologys.com/selenium-training-in-chennai.html

Step 2:
-------

Unzip the downloaded ZIP file and copy the subfolder "cmdline-tools".
Then, create a folder in the C drive following this path:

C:/softwares/android_sdk

Paste the "cmdline-tools" folder inside "android_sdk".

Step 3:
-------

Create a new folder inside "cmdline-tools" and name it "latest".

Step 4:
-------

Cut the following files and folders from "cmdline-tools":

- bin
- lib
- NOTICE
- source.properties

Paste them inside the "latest" folder you created in Step 3.

Step 5:
-------

Set the System Environment Variable to include the bin path for accessing sdkmanager.

Step 6:
-------

Run the following command to install platform-tools and the required platform package:

sdkmanager "platform-tools" "platforms;android-33"

Step 7:
-------

When prompted in the Command Prompt, type Y and press Enter to accept the licenses and proceed with the installation.

Wait for the installation to complete. Once done, the platform-tools and platform package will be successfully installed.

Step 8:
-------

Verify that the platform-tools and platforms folders have been created in the android_sdk folder.

Step 9:
-------

Create the ANDROID_HOME environment variable to set the Android SDK folder as the home path.

Step 10:
--------

Set the platform-tools path in the environment variable so you can access adb from anywhere.

Step 11:
--------

Open Command Prompt and execute the ADB command to verify that the Android Debug Bridge (ADB) is working correctly.

Step 12:
--------

Verify that you see ADB options to confirm that the ADB command is working properly.

Appium Installation Commands:
-----------------------------

Step 1:
-------

Open Command Prompt after successful Node.js installation, and run the following command to install Appium:

npm install -g Appium

Step 2:
-------

Wait until the installation process completes. Once it is done, verify the installation by running the following command:

appium -v

UI Automator2 Installation:
---------------------------

Step 1:
-------

To install the UIAutomator2 driver for Appium, run the following command in Command Prompt:

appium driver install uiautomator2

This will install the UIAutomator2 driver, which is used for automating Android devices.

Step 2:
-------

After the installation completes, verify the installed drivers by running the following command:

appium driver list

This command will display the list of available Appium drivers, including UIAutomator2 if it was installed successfully.

Launch App through Automation:
------------------------------

Required dependency:
--------------------

-> java-client


Sample Code:
------------

import java.net.URL;
import org.openqa.selenium.remote.DesiredCapabilities;
import io.appium.java_client.android.AndroidDriver;

public class LaunchApp {

    public static void main(String[] args) throws Exception {

        DesiredCapabilities caps = new DesiredCapabilities();

        // Mobile platform
        caps.setCapability("platformName", "Android");

        // Device name
        caps.setCapability("deviceName", "Android Device");

        // Automation engine
        caps.setCapability("automationName", "UiAutomator2");

        // App package
        caps.setCapability("appPackage", "com.android.settings");

        // App activity
        caps.setCapability("appActivity", "com.android.settings.Settings");

        // Appium server URL
        URL url = new URL("http://127.0.0.1:4723");

        // Start driver session
        AndroidDriver driver = new AndroidDriver(url, caps);

        System.out.println("App Launched Successfully");

    }
}


URL - Connect to Appium Server

DesiredCapabilities - Define device & app details

AndroidDriver - Communicate with Android device
